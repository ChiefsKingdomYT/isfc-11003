    • Progressive Enhancement

      Reading Aaron Gustafson’s "Understanding Progressive Enhancement" really put the semester into perspective. He uses the analogy of a peanut—the HTML is the nut at the center, the CSS is the shell, and JavaScript is the candy coating.

      Basically, it taught me that a website should work even if the "fancy" stuff breaks which tends to happen a lot. When I was working on my Mars Rover project, I made sure my HTML structure was solid first. That way, even if someone is using a weird browser where my CSS Grid doesn't load right, they can still actually read about the mission. It’s about building a site that's accessible to everyone, not just people with the newest tech.

      Mars Rover: Final Thoughts & Revisions

      What I Changed (Feedback)

      I had a friend look at my first attempt, and they were pretty honest about what wasn't working. On their phone, the rover images were way too big and kept overlapping the text, which looked messy. Also, the buttons I made for the slideshow were tiny and hard to tap.

      To fix this, I went back into the CSS and used Flexbox to make everything responsive. I also sized up the buttons so they're actually usable on mobile. I kept my original dark theme because I liked the "space" vibe, but I cleaned up the spacing so it doesn't feel as cramped.

      What I Learned

      Looking back, I’ve come a long way from those first HTML prototypes. I’m actually pretty proud of getting the CSS Grid and the slideshow logic to work—that was a bit of a headache at first.

      If I had to start over, I’d definitely go "Mobile-First." I spent way too much time at the end of the project trying to fix layout glitches on smaller screens. Starting small and building up would have saved me a lot of stress. Overall, seeing all these labs—from the Floating exercises to the Flexbox Froggy game—finally come together in one big project was a cool way to end the semester. I really enjoyed the class, once i started being able to stay awake after consitently taking my iron pills and wearing my glasses for astigmatism it was an enjoyable class
